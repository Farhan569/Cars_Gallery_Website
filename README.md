# Cars Gallery Website

A static frontend website that displays a gallery of cars along with their basic details. The project focuses on layout, structure, and presentation of car information using plain web technologies.

## Features

- Car gallery with images  
- Car detail sections (name, model, description, specs)  
- Responsive layout  
- Simple and clean UI  
- Fully static (no backend)

## Technologies Used

- HTML  
- CSS  
- JavaScript  

No frameworks. No backend. No database.

## How to Run

1. Clone the repository:

```
git clone https://github.com/Farhan569/Cars_Gallery_Website/tree/main
```

2. Open `index.html` in a browser  
or run a local server.

## Deployment

This project can be deployed using:
- GitHub Pages  
- Netlify  
- Vercel  

## Limitations

- Static data only  
- No authentication  
- No API integration  
- No backend logic  

## Purpose

This project was built to practice frontend development and demonstrate structured presentation of static data.
